# Twitter Bootstrap 4 Alpha - Tests

## First Setup

	bower install
	npm install
	
## Run the project

	npm start
	
## Compile SASS

	npm run build
	
	
## Development Session

Open two terminal windows (or tabs) and run

	// terminal 1
	npm run watch
	
	// terminal 2
	npm start
	
NOTE: when the process start it fucks up the css, just save the scss file the first time and it will be ok.
